# Sprint Roulette 🎯

スプリントレビューの発表者をランダムに選出するルーレットツールです。

## 機能

- **ルーレット**: ボタンを押すとランダムに発表者を選出
- **タイマー**: 10分のカウントダウン（残り2分で黄色、30秒で赤色に変化）
- **メンバー編集**: 右上の「⚙️ メンバー編集」から追加・削除可能

## 使い方

1. GitHub Pages の URL にアクセス
2. 「ルーレットを回す」ボタンをクリック
3. 発表者が決まったら「スタート」でタイマー開始
4. 次の発表者を選ぶときは再度ルーレットを回す
5. 全員終わったら「リセット」で最初から

## メンバーの変更

### 画面から変更

1. 右上の「⚙️ メンバー編集」をクリック
2. メンバーを追加・削除
3. 「保存して閉じる」

> **注意**: メンバーリストはブラウザの localStorage に保存されます。
> スプリントレビューの担当者が毎回変わる場合、各担当者が自分のブラウザで初回にメンバー設定を行う必要があります。

### デフォルトメンバーを変更（全員に反映）

`index.html` 内の `DEFAULT_MEMBERS` を編集してコミットしてください：

```javascript
const DEFAULT_MEMBERS = ["メンバー1", "メンバー2", "メンバー3"];
```

チームメンバーの増減があった場合はこちらを更新すると、全員の初期値に反映されます。

## デバッグモード

`D` キーを押すとデバッグパネルが表示されます。
